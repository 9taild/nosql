Below are examples of **5 to 10 operations for Read, Update, and Delete**: --- ### **Read Operations** 1. **Read All Data**: ```cql SELECT * FROM students; ``` 2. **Read Specific Columns**: ```cql SELECT name, age FROM students; ``` 3. **Read Data by Specific Condition**: ```cql SELECT * FROM students WHERE course='Mathematics'; ``` 4. **Read Data by Multiple Conditions**: ```cql SELECT * FROM students WHERE age > 22 AND course='Physics'; ``` 5. **Read Data with a LIMIT**: ```cql SELECT * FROM students LIMIT 5; ``` --- ### **Update Operations** 1. **Update Single Column**: ```cql UPDATE students SET age = 23 WHERE id = ; ``` 2. **Update Multiple Columns**: ```cql UPDATE students SET age = 24, course = 'Computer Engineering' WHERE id = ; ``` 3. **Update Data Based on Specific Condition**: ```cql UPDATE students SET course = 'Data Science' WHERE name = 'John Doe'; ``` 4. **Update Data for a Specific Student**: ```cql UPDATE students SET name = 'Charlie Brown' WHERE id = ; ``` 5. **Update Multiple Rows (based on a condition)**: ```cql UPDATE students SET course = 'Physics' WHERE age > 23; ``` --- ### **Delete Operations** 1. **Delete Specific Row**: ```cql DELETE FROM students WHERE id = ; ``` 2. **Delete Data by Condition**: ```cql DELETE FROM students WHERE name = 'John Doe'; ``` 3. **Delete All Rows with Specific Condition**: ```cql DELETE FROM students WHERE course = 'Computer Science'; ``` 4. **Delete Specific Column**: ```cql DELETE age FROM students WHERE id = ; ``` 5. **Delete All Data (for a specific student)**: ```cql DELETE FROM students WHERE name = 'Bob Brown'; ``` 6. **Delete All Rows in Table**: To delete all data from a table, truncate it (remove all rows): ```cql TRUNCATE students; ``` --- ### **Additional Operations** 1. **Count Rows in Table**: ```cql SELECT COUNT(*) FROM students; ``` 2. **Create Index on a Column**: ```cql CREATE INDEX ON students (course); ``` 3. **Check Schema (Keyspace & Table Details)**: ```cql DESCRIBE KEYSPACES; DESCRIBE TABLE students; ``` 4. **Drop a Table**: ```cql DROP TABLE students; ``` --- ### **Final Notes** - **Read Operations** allow you to fetch data based on specific columns or conditions. - **Update Operations** change existing data for a given row identified by the primary key. - **Delete Operations** remove rows or individual columns from the table.
